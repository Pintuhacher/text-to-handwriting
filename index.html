Everything you need to know

How can I add my own handwriting?

You can create a font from your handwriting using services like Calligraphr. Once you have a .ttf file, upload it using the "Upload Custom Font" option.

Where can I get more fonts?

You can find additional handwriting fonts at Quantum Enterprises Font Vault or other font repositories online.

Why are there gaps between letters?

This can happen with some custom fonts. Use the letter spacing and word spacing controls in the customization panel to adjust the gaps and improve readability.

Can I use this for commercial purposes?

The tool itself is free to use, but make sure you have proper licensing for any custom fonts you upload. Always check font licenses before commercial use.
